# FAQ-CMS

B2B SaaS型のFAQ管理システム

## 概要

FAQ-CMSは、中小〜中堅企業のCS/マーケティング部門向けに設計されたシンプルで実用的なFAQ管理システムです。Amber（琥珀色）をベースとした温かみのあるデザインが特徴です。

## 主な機能

### 公開サイト
- トップページ（カテゴリ一覧）
- カテゴリ詳細ページ
- 記事詳細ページ（セクションカード型表示）
- 検索機能（サジェスト付き）
- 問い合わせフォーム
- フィードバック機能

### 管理画面
- ダッシュボード（KPI表示）
- 記事管理（一覧・作成・編集）
- セクションカード型エディタ
- カテゴリ管理（階層構造対応）
- タグ管理
- ユーザー管理
- アクセス分析
- ヒット0クエリ分析
- フィードバック管理
- サイト設定

## 技術スタック

- **フレームワーク**: React + TypeScript
- **スタイリング**: Tailwind CSS v4
- **ビルドツール**: Vite
- **アイコン**: Lucide React
- **パッケージマネージャー**: pnpm

## セットアップ

```bash
# 依存関係のインストール
pnpm install

# 開発サーバーの起動
pnpm dev

# ビルド
pnpm build
```

## デザインコンセプト

- **シンプル・実用優先**: 装飾を最小限に、情報設計重視
- **温かみのある親しみやすさ**: Amber色で柔らかい印象
- **フラット・クリーン**: グラデーション・シャドウは最小限
- **B2B SaaSとしての信頼性**: プロフェッショナルな見た目

## カラーパレット

- Primary: Amber (#BA7517, #EF9F27)
- Accent: Coral (#D85A30)
- Neutral: Gray (#2C2C2A, #5F5E5A, #D3D1C7)
